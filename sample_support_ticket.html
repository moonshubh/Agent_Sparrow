I see the issue now - it's an authentication problem. This usually happens when:

  • Your password has changed recently
  • 2-factor authentication is enabled
  • App-specific passwords are required

Please try this:

  1. Go to your Google Account settings
  2. Enable "Less secure app access" OR create an App Password
  3. Use the App Password in Mailbird instead of your regular password

This should resolve the authentication error. Let me know how it goes!
