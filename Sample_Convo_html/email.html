1. Click the Start button and type cmd.

3.

ipconfig /flushdns
4. You should see a message confirming the DNS Resolver Cache was successfully flushed.
5. Close the command prompt and restart your device. Check if the issue is resolved.

Step 2: Change Your DNS Server (If Step 1 Doesn't Work)

Sometimes, the default DNS server provided by your Internet Service Provider (ISP) can be slow or have issues. Switching to a public DNS server like Google's is a great troubleshooting step.

1. Press the Windows Key + R to open the Run dialog.

3. Right-click on the network adapter you are currently using (e.g., "Wi-Fi" or "Ethernet") and select Properties.
